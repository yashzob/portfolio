Project Summary Report

Rent the Runway Data Science Project

Project Overview

The Rent the Runway project is a data science analysis aimed at understanding customer and product characteristics from rental data. The project involves data cleaning, feature engineering, predictive modeling, dimensionality reduction, and clustering to extract actionable insights for business decision-making.

Technology Stack

  • Programming Language: Python
  • Libraries: pandas, numpy, matplotlib, seaborn, scikit-learn, nltk
  • Tools: Jupyter Notebook, NLTK Sentiment Analyzer

Data and Architecture

  • Dataset: Rent the Runway rental data (renttherunway.csv) containing customer demographics, product details, rental occasions, and reviews.
  • Data preprocessing includes handling missing values, converting categorical variables, and feature engineering such as height conversion and bust size encoding.
  • Predictive modeling is used to impute missing weight values.
  • Dimensionality reduction via PCA reduces feature space complexity.
  • Clustering algorithms (KMeans and Agglomerative) segment customers/products based on derived features.

Features and Functionalities

  • Data cleansing: Removal of duplicates, redundant columns, and handling of inconsistent data formats.
  • Feature engineering: Conversion of height to inches, numerical encoding of bust size, sentiment analysis on review text.
  • Missing data imputation: Predictive modeling for weight, mean/median imputation for other features.
  • Exploratory data analysis: Statistical summaries, outlier detection and treatment.
  • Dimensionality reduction: PCA to identify principal components explaining variance.
  • Clustering: Identification of customer/product segments using KMeans and Agglomerative clustering with evaluation metrics.

Model Evaluation and Insights

  • AdaBoost regression model used for weight prediction with evaluation by mean squared error and accuracy within tolerance.
  • PCA results indicated limited linear dependencies among features.
  • Clustering revealed 4-5 distinct segments, useful for targeted marketing and inventory decisions.
  • Sentiment analysis provided additional insights into customer reviews.

Impact

This project transforms raw rental data into meaningful insights, enabling Rent the Runway to better understand customer segments, improve inventory management, and tailor marketing strategies. The comprehensive data science workflow ensures data quality and robust analysis, supporting data-driven business decisions.
